# Getting Started

## Requirements

- [Gatsby CLI](https://www.npmjs.com/package/gatsby-cli)
- [Yarn](https://yarnpkg.com/en/)

- Install the Gatsby CLI globally:

```
yarn global add gatsby-cli
```

- Navigate to your new directory and run:

```
yarn develop
```
